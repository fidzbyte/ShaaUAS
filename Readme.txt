Cara Menjalankan Database db_rs dari rekammedik.sql
Buka XAMPP, klik Start pada Apache dan MySQL

Buka browser: http://localhost/phpmyadmin

Klik Database → buat database baru bernama db_rs

Klik db_rs → tab Import → pilih file rekammedik.sql → klik Go

Selesai! Database siap digunakan di localhost
